# ECN-004: axisFormat プリセットドロップダウン

- **ステータス**: 適用済
- **種別**: 改善

- **対象コミット**: `74080ec`, `fe004c3`
- **影響ファイル**: `mermaid-assist.html`, `tests/e2e/brushup-evaluation.spec.js`

## コンテキスト

`axisFormat` (X軸の日付フォーマット) は d3-time-format 記法（`%m/%d`, `%Y-%m-%d`等）を直接入力する必要があり、初学者には記法を覚えるコストが高かった。

## 対策

7つのプリセット（プレビュー付き）+ 「カスタム…」選択のドロップダウンに置換:

| 表示 | 内部値 |
|---|---|
| `2026-04-14` (年-月-日) | `%Y-%m-%d` |
| `2026/04/14` | `%Y/%m/%d` |
| `04/14` | `%m/%d` |
| `04月14日` | `%m月%d日` |
| `Apr 14` | `%b %d` |
| `14 Apr` | `%d %b` |
| `Tue` (曜日) | `%a` |
| カスタム… | テキスト入力欄を表示 |

エディタで非プリセット値に書き換えると自動で「カスタム」が選ばれて現在値表示。
