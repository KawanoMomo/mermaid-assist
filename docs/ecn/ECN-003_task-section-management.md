# ECN-003: タスク順序操作・セクション管理 UI

- **ステータス**: 適用済
- **種別**: 機能追加
- **バージョン**: v0.3.0
- **対象コミット**: `d103957` ... `2835952`
- **影響ファイル**: `mermaid-assist.html`, `tests/e2e/brushup-evaluation.spec.js`

## コンテキスト

ガントチャート編集において、タスクの並び順変更とセクション間移動が GUI から不可能だった。また、セクション削除のロジックは存在したが GUI からアクセスする手段がなかった。

## 対策

3機能を追加（評価項目 E11-E13）:

- **E11**: 同一セクション内でタスクを上下移動する `↑↓` ボタンを property panel に追加
- **E12**: タスクを別セクションへ移動するドロップダウンを property panel に追加（移動先 = 末尾）
- **E13**: 未選択時 panel にセクション一覧を表示。各セクションに削除ボタン（タスクごと削除、確認ダイアログ付き）

タスク上下移動はセクション境界を超えない方針で誤操作を防止。セクション間移動はドロップダウン経由で明示的に行う。

## 結果

- 81 → 87 E2E テスト（6件追加）
- v0.3.0 リリース
